Python

  • Início

O Python possui uma vasta gama de comandos, funções e palavras-chave. Abaixo está uma lista organizada por categorias:

1. Palavras-chave (Keywords)

Essas são palavras reservadas em Python e não podem ser usadas como identificadores (nomes de variáveis, funções, etc.):


    False      await        else        import       pass
    None       break        except      in           raise
    True       class        finally     is           return
    and        continue     for         lambda       try
    as         def          from        nonlocal     while
    assert     del          global      not          with
    async      elif         if          or           yield

2. Funções Embutidas (Built-in Functions)

Manipulação de Dados e Conversões

  • abs() - Retorna o valor absoluto.
  • bin() - Converte um número para binário.
  • bool() - Converte para booleano.
  • bytes() - Cria um objeto bytes.
  • chr() - Retorna o caractere Unicode correspondente.
  • complex() - Cria números complexos.
  • dict() - Cria um dicionário.
  • float() - Converte para ponto flutuante.
  • hex() - Converte um número para hexadecimal.
  • int() - Converte para inteiro.
  • list() - Cria uma lista.
  • oct() - Converte para octal.
  • ord() - Retorna o código Unicode de um caractere.
  • range() - Gera uma sequência de números.
  • round() - Arredonda um número.
  • set() - Cria um conjunto.
  • str() - Converte para string.
  • tuple() - Cria uma tupla.
  • type() - Retorna o tipo de um objeto.

Operações Matemáticas

  • divmod() - Retorna o quociente e o resto.
  • max() - Maior valor em um iterável.
  • min() - Menor valor em um iterável.
  • pow() - Eleva um número a uma potência.
  • sum() - Soma os elementos de um iterável.

Manipulação de Sequências

  • all() - Verifica se todos os elementos são verdadeiros.
  • any() - Verifica se algum elemento é verdadeiro.
  • enumerate() - Enumera os elementos de um iterável.
  • filter() - Filtra elementos com base em uma condição.
  • len() - Retorna o comprimento de um objeto.
  • map() - Aplica uma função a todos os itens de um iterável.
  • sorted() - Ordena elementos.
  • zip() - Agrupa elementos de iteráveis.

Entrada e Saída

  • input() - Lê dados do usuário.
  • print() - Exibe informações no console.
  • open() - Abre um arquivo.

Manipulação de Arquivos

  • close() - Fecha um arquivo.
  • read() - Lê o conteúdo de um arquivo.
  • write() - Escreve em um arquivo.
  • readline() - Lê uma linha do arquivo.

Programação Funcional

  • lambda - Cria funções anônimas.
  • map() - Aplica uma função a todos os elementos de um iterável.
  • filter() - Filtra elementos com base em uma condição.
  • reduce() - Aplica uma função cumulativa (importado de functools).

Funções de Depuração e Testes

  • assert - Verifica se uma condição é verdadeira.
  • help() - Mostra documentação sobre um objeto.
  • dir() - Lista atributos e métodos disponíveis.
  • id() - Retorna o identificador de um objeto.
  • repr() - Representação em string de um objeto.

3. Estruturas de Controle de Fluxo

Decisões (Condicional)

    if condicao:
        # código
    elif outra_condicao:
        # código
    else:
        # código

Laços de Repetição (Loops)

    for i in range(10):
        # código

    while condicao:
        # código

Controle de Loop

  • break - Interrompe o loop.
  • continue - Pula para a próxima iteração.
  • pass - Passa sem executar nada.

4. Estruturas de Dados

  • Listas: [1, 2, 3] - Coleção ordenada e mutável.
  • Tuplas: (1, 2, 3) - Coleção ordenada e imutável.
  • Conjuntos: {1, 2, 3} - Coleção não ordenada e sem duplicatas.
  • Dicionários: {"chave": "valor"} - Pares chave-valor.

5. Funções e Classes

Definindo Funções:

    def minha_funcao(param1, param2):
        return param1 + param2

Classes e Objetos:

    class Pessoa:
        def __init__(self, nome):
            self.nome = nome

        def saudacao(self):
            print(f"Olá, {self.nome}!")

6. Tratamento de Exceções

    try:
        x = 10 / 0
    except ZeroDivisionError:
        print("Erro: Divisão por zero.")
    finally:
        print("Finalizado.")

7. Módulos e Pacotes

  • Importação de Módulos:
    import math
    from random import randint
    import os
  • Principais Módulos Integrados:
    • math - Funções matemáticas avançadas.
    • random - Geração de números aleatórios.
    • datetime - Manipulação de datas e horas.
    • os - Interação com o sistema operacional.
    • sys - Manipulação do sistema e argumentos.
    • re - Expressões regulares.
    • json - Manipulação de JSON.
    • csv - Leitura e escrita em arquivos CSV.
    • sqlite3 - Banco de dados SQL.

8. Manipulação Avançada

  • Compreensão de Listas (List Comprehension):
    quadrados = [x**2 for x in range(10)]
  • Geradores:
    gen = (x**2 for x in range(10))
  • Expressões Lambda:
    soma = lambda a, b: a + b
