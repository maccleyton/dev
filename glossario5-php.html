Aqui está uma lista abrangente dos comandos, funções e palavras-chave possíveis no PHP, organizados por categorias:

1. Palavras-chave Reservadas (Keywords)

Essas palavras são reservadas e não podem ser usadas como identificadores (nomes de variáveis, funções, classes, etc.):


    abstract       and           as               break            case
    catch          class         clone            const            continue

    endwhile       extends       final            finally          for
    foreach        function      global           goto             if
    implements     include       include_once     instanceof       insteadof
    interface      namespace     new              or               private
    protected      public        require          require_once     return

2. Operadores

Operadores Aritméticos:

  • + - Adição
  • - - Subtração
  • * - Multiplicação
  • / - Divisão
  • % - Módulo
  • ** - Exponenciação

Operadores de Comparação:

  • == - Igual
  • === - Idêntico (valor e tipo)
  • != ou <> - Diferente
  • !== - Não idêntico
  • < - Menor que
  • > - Maior que
  • <= - Menor ou igual
  • >= - Maior ou igual

Operadores Lógicos:

  • && - E lógico
  • || - Ou lógico
  • ! - Não lógico
  • and - E lógico com menor precedência
  • or - Ou lógico com menor precedência
  • xor - Ou exclusivo

Operadores de Atribuição:

  • = - Atribuição
  • += - Adição e atribuição
  • -= - Subtração e atribuição
  • *= - Multiplicação e atribuição
  • /= - Divisão e atribuição
  • %= - Módulo e atribuição

Outros Operadores:

  • . - Concatenação de strings
  • => - Associação em arrays
  • ?? - Operador de coalescência nula
  • ?: - Operador ternário

3. Estruturas de Controle

Condicionais:

    if ($x > 0) {
        echo "Positivo";
    } elseif ($x < 0) {
        echo "Negativo";
    } else {
        echo "Zero";
    }

Switch:

    switch ($x) {
        case 1:
            echo "Um";
            break;
        case 2:
            echo "Dois";
            break;
        default:
            echo "Outro";
    }

Laços de Repetição:

    for ($i = 0; $i < 10; $i++) {
        echo $i;
    }

    while ($i < 10) {
        echo $i++;
    }

    do {
        echo $i++;
    } while ($i < 10);

    foreach ($array as $key => $value) {
        echo "$key => $value";
    }

4. Funções para Manipulação de Strings

  • strlen() - Retorna o comprimento de uma string.
  • strpos() - Procura a posição de uma substring.
  • substr() - Extrai uma parte de uma string.
  • str_replace() - Substitui partes de uma string.
  • trim() - Remove espaços em branco.
  • explode() - Divide uma string em array.
  • implode() - Junta elementos de um array em string.
  • strtolower() - Converte para minúsculas.
  • strtoupper() - Converte para maiúsculas.

5. Funções para Arrays

  • array() - Cria um array.
  • count() - Conta os elementos.
  • array_push() - Adiciona elementos no final.
  • array_pop() - Remove o último elemento.
  • array_shift() - Remove o primeiro elemento.
  • array_unshift() - Adiciona elementos no início.
  • array_merge() - Mescla arrays.
  • array_key_exists() - Verifica se uma chave existe.
  • in_array() - Verifica se um valor existe no array.
  • sort() - Ordena os valores.
  • rsort() - Ordena em ordem decrescente.
  • asort() - Ordena preservando as chaves.
  • ksort() - Ordena pelas chaves.

6. Manipulação de Arquivos

  • fopen() - Abre um arquivo.
  • fclose() - Fecha um arquivo.
  • fwrite() - Escreve em um arquivo.
  • fread() - Lê o conteúdo de um arquivo.
  • file_get_contents() - Lê todo o conteúdo de um arquivo.
  • file_put_contents() - Escreve em um arquivo.
  • unlink() - Exclui um arquivo.

7. Manipulação de Datas

  • date() - Formata a data e hora.
  • time() - Retorna o timestamp atual.
  • strtotime() - Converte string para timestamp.
  • mktime() - Cria um timestamp.

8. Funções para Manipulação de Variáveis

  • isset() - Verifica se a variável foi definida.
  • unset() - Remove uma variável.
  • empty() - Verifica se está vazia.
  • is_array() - Verifica se é array.
  • is_numeric() - Verifica se é numérico.
  • is_string() - Verifica se é string.
  • is_object() - Verifica se é objeto.
  • gettype() - Retorna o tipo de variável.
  • settype() - Define o tipo de variável.

9. Funções para Banco de Dados (MySQLi)

  • mysqli_connect() - Conecta ao banco de dados.
  • mysqli_query() - Executa uma query.
  • mysqli_fetch_assoc() - Obtém resultados como array associativo.
  • mysqli_fetch_array() - Obtém resultados como array.
  • mysqli_num_rows() - Retorna o número de linhas.
  • mysqli_close() - Fecha a conexão.

10. Classes e Objetos

Definindo Classes:

    class Pessoa {
        public $nome;

        public function __construct($nome) {
            $this->nome = $nome;
        }

        public function saudacao() {
            return "Olá, " . $this->nome;
        }
    }

    $pessoa = new Pessoa("João");
    echo $pessoa->saudacao();

11. Tratamento de Exceções

    try {
        throw new Exception("Erro!");
    } catch (Exception $e) {
        echo $e->getMessage();
    } finally {
        echo "Finalizado.";
    }
