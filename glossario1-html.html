HTML - Hypertext Markup Language

  • Início

Aqui está uma lista dos comandos e elementos disponíveis no HTML:

1. Estrutura Básica do HTML

  • <!DOCTYPE html> - Declara o tipo de documento.
  • <html> - Define o elemento raiz do documento.
  • <head> - Contém metadados e links externos.
  • <title> - Define o título da página.
  • <body> - Contém o conteúdo visível da página.

2. Elementos de Texto e Formatação

  • Cabeçalhos:

    • <h1> até <h6> - Títulos, sendo <h1> o maior.

  • Parágrafos e Quebras:

    • <p> - Parágrafo.
    • <br> - Quebra de linha. (sem fechamento)
    • <hr> - Linha horizontal. (sem fechamento)

  • Formatação de Texto:

    • <b> - Texto em negrito.
    • <strong> - Texto com ênfase (semântico, importante).
    • <i> - Texto em itálico.
    • <em> - Texto enfatizado (semântico).
    • <u> - Texto sublinhado.
    • <mark> - Texto destacado.
    • <small> - Texto menor.
    • <sup> - Texto sobrescrito.
    • <sub> - Texto subscrito.
    • <del> - Texto riscado.
    • <ins> - Texto inserido/sublinhado.
    • <code> - Texto formatado como código.
    • <pre> - Texto pré-formatado.

3. Links e Âncoras

  • <a> - Hiperlink (com atributo href).
    Exemplo: <a href="https://exemplo.com">Clique aqui</a>
  • target - Define onde abrir o link (_blank para nova aba).
  • <link> - Conecta recursos externos, como folhas de estilo CSS. (Usado no <head>)

4. Listas

  • Listas Não Ordenadas:

    • <ul> - Lista não ordenada.
    • <li> - Item de lista.

  • Listas Ordenadas:

    • <ol> - Lista ordenada.
    • <li> - Item de lista.
    • type - Define o estilo (1, A, a, I, i).

  • Listas de Definição:

    • <dl> - Lista de definição.
    • <dt> - Termo da definição.
    • <dd> - Descrição do termo.

5. Imagens e Multimídia

  • <img> - Insere uma imagem.
    Atributos importantes:

    • src - Caminho da imagem.
    • alt - Texto alternativo para acessibilidade.
    • width e height - Tamanho.

  • Áudio e Vídeo:

    • <audio> - Adiciona áudio.
    • <video> - Adiciona vídeo.
    • <source> - Fonte do áudio/vídeo.
    • controls - Mostra controles de reprodução.

6. Tabelas

  • Estrutura Básica:

    • <table> - Define a tabela.
    • <tr> - Linha da tabela.
    • <th> - Cabeçalho da tabela.
    • <td> - Célula da tabela.

  • Propriedades Avançadas:

    • <caption> - Legenda da tabela.
    • <thead> - Cabeçalho agrupado.
    • <tbody> - Corpo da tabela.
    • <tfoot> - Rodapé da tabela.
    • colspan - Mesclar colunas.
    • rowspan - Mesclar linhas.

7. Formulários e Inputs

  • Estrutura:

    • <form> - Cria o formulário.
    • <input> - Campo de entrada.
    • <label> - Rótulo associado a um campo.
    • <textarea> - Caixa de texto multilinhas.
    • <select> - Menu suspenso.
    • <option> - Opção dentro do <select>.
    • <button> - Botão clicável.

  • Tipos de Input:

    • text, password, email, number, tel, url, date, time, checkbox, radio, range, file, color, submit, reset, hidden.

8. Estrutura Semântica (HTML5)

  • Seções e Agrupamentos:
    • <header> - Cabeçalho de seção.
    • <nav> - Navegação.
    • <main> - Conteúdo principal.
    • <section> - Seção de conteúdo.
    • <article> - Artigo de conteúdo independente.
    • <aside> - Conteúdo relacionado (barra lateral).
    • <footer> - Rodapé.
    • <figure> - Agrupa mídia como imagens.
    • <figcaption> - Legenda para <figure>.

Esses elementos ajudam a criar uma estrutura HTML bem organizada e semântica, tornando o site mais acessível e fácil de entender tanto para usuários quanto para motores de busca (SEO).

<main>

  • Definição: Representa o conteúdo principal de uma página. Esse conteúdo deve ser único e relevante para o propósito principal da página.
  • Características:
    • Deve conter apenas um único <main> por página.
    • Exclui cabeçalhos, rodapés, barras laterais e outros elementos que não fazem parte do "conteúdo principal".
    • Facilita a navegação para tecnologias assistivas, como leitores de tela.

Exemplo:

    <main>
            <h1>Bem-vindo ao meu site</h1>
            <p>Este é o conteúdo principal da página.</p>
        </main>

<header>

  • Definição: Representa o cabeçalho de um documento ou de uma seção. Pode conter títulos, logotipos, menus de navegação, etc.
  • Características:
    • Pode ser usado várias vezes em uma página, por exemplo, no cabeçalho principal da página e nos cabeçalhos de seções ou artigos.
    • É ideal para introduzir o conteúdo de uma página ou seção.

Exemplo:

    <header>
            <h1>Minha Página</h1>
            <nav>
                <ul>
                    <li><a href="#inicio">Início</a></li>
                    <li><a href="#sobre">Sobre</a></li>
                    <li><a href="#contato">Contato</a></li>
                </ul>
            </nav>
        </header>

<footer>

  • Representa o rodapé de um documento ou seção. Pode conter informações como direitos autorais, links para políticas, rodapés de seção, etc.
  • Pode ser usado múltiplas vezes.

Exemplo:

    <footer>
            <p>© 2025 Minha Empresa. Todos os direitos reservados.</p>
        </footer>

<section>

  • Representa uma seção temática de um documento. É ideal para agrupar conteúdo relacionado.
  • Pode conter cabeçalhos, parágrafos, imagens e outros elementos.

Exemplo:

    <section>
            <h2>Sobre Nós</h2>
            <p>Somos uma empresa dedicada à tecnologia.</p>
        </section>

<article>

  • Representa conteúdo independente e autocontido, como uma postagem de blog, um artigo de notícias ou um comentário.
  • Pode ser compartilhado ou reutilizado fora de seu contexto original.

Exemplo:

    <article>
            <h2>Postagem no Blog</h2>
            <p>Este é o conteúdo do artigo.</p>
        </article>

<aside>

  • Representa conteúdo complementar ou relacionado ao principal, como barras laterais, caixas de informações ou anúncios.

Exemplo:

    <aside>
            <h3>Mais informações</h3>
            <p>Confira nossos outros artigos.</p>
        </aside>

<nav>

  • Representa um conjunto de links de navegação principais da página.

Exemplo:

    <nav>
            <ul>
                <li><a href="#inicio">Início</a></li>
                <li><a href="#sobre">Sobre</a></li>
                <li><a href="#contato">Contato</a></li>
            </ul>
        </nav>

Resumo das Diferenças

Elemento Função Pode ser usado várias vezes?
<main> Conteúdo principal da página, único e central. Não
<header> Cabeçalho de uma página ou seção, com título e navegação. Sim
<footer> Rodapé de uma página ou seção. Sim
<section> Agrupa conteúdo tematicamente relacionado. Sim
<article> Conteúdo autocontido, como postagens de blog ou notícias. Sim
<aside> Conteúdo relacionado ao principal, mas não essencial (barras laterais). Sim
<nav> Links de navegação principais da página. Sim

9. Tags Interativas e Scripts

  • Multimídia e Interatividade:

    • <canvas> - Gráficos e desenhos.
    • <svg> - Gráficos vetoriais escaláveis.
    • <iframe> - Incorporar páginas.
    • <embed> - Conteúdos externos (vídeos, PDFs).
    • <object> - Objetos externos.
    • <param> - Parâmetro para objetos.

  • Scripts:

    • <script> - Código JavaScript embutido ou externo.
    • <noscript> - Alternativa quando JavaScript está desativado.

10. Metadados e SEO

  • No <head>:
    • <meta> - Informações de metadados.
    • <meta charset="UTF-8"> - Codificação do texto.
    • <meta name="viewport"> - Configurações para dispositivos móveis.
    • <meta name="description"> - Descrição para SEO.
    • <meta name="keywords"> - Palavras-chave para SEO.
    • <meta name="author"> - Autor do site.

11. Tags Obsoletas (Evitar o Uso)

  • <center> - Substituído por CSS.
  • <font> - Substituído por CSS.
  • <marquee> - Uso descontinuado.
  • <big> - Substituído por CSS.
