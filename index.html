1. Linguagem HTML

Definição:

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas da webe estruturar seu conteúdo. Ele define a estrutura e o conteúdo das páginas, utilizando uma série de elementos chamados tags. Essas tags indicam ao navegador como o conteúdo deve ser exibido.

Por exemplo:

  • Estrutura básica de um documento HTML:
    <!DOCTYPE html>
    <html>
    <head>
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Bem-vindo ao meu site!</h1>
        <p>Este é um parágrafo de exemplo.</p>
    </body>
    </html>

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

“Hello World” na linguagem HTML

    <!DOCTYPE html>
    <html>
    <head>
        <title>Hello, World</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
    </html>

Componentes principais do HTML:

  1. Tags: Delimitam o início e o fim de elementos (exemplo: <h1> e </h1>).
  2. Atributos: Adicionam informações extras às tags (exemplo: <img src="imagem.jpg" alt="Descrição">).
  3. Conteúdo: O texto ou elementos aninhados entre as tags (exemplo: Bem-vindo ao meu site! no <h1>).

Exemplos de tags comuns:

  • Estruturais: <html>, <head>, <body>.
  • Títulos: <h1> a <h6> (cabeçalhos de diferentes tamanhos).
  • Texto: <p> (parágrafo), <strong> (negrito), <em> (itálico).
  • Links e imagens: <a> (link), <img> (imagem).
  • Listas: <ul> (não ordenada), <ol> (ordenada), <li> (itens da lista).
  • Tabelas: <table>, <tr>, <td>.

Por que o HTML é importante?

Ele é a base de qualquer página da web e, junto com CSS (responsável pelo design) e JavaScript (responsável por interatividade), forma a base do desenvolvimento web.

2. Linguagem CSS

Definição:

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML (incluindo suas variantes, como XHTML). Ele controla como os elementos da página serão exibidos visualmente no navegador, permitindo separar o conteúdo (HTML) do design (CSS).

CSS é fundamental no desenvolvimento web, pois transforma páginas simples em experiências visuais ricas e dinâmicas.

Como o CSS funciona?

O CSS trabalha com "regras" aplicadas aos elementos do HTML. Cada regra consiste em:

  1. Seletor: Indica qual elemento será estilizado (exemplo: p para parágrafos).
  2. Declarações: São compostas por propriedades e valores, definindo os estilos desejados (exemplo: color: blue; para definir a cor do texto como azul).

Exemplo de CSS:

    p {
        color: blue; /* Define a cor do texto como azul */
        font-size: 16px; /* Define o tamanho da fonte */
    }

Quando esse código CSS é aplicado a um parágrafo em HTML, ele altera sua aparência conforme as regras especificadas.

Formas de usar CSS:

  1. Inline: O estilo é adicionado diretamente no elemento HTML usando o atributo style. 
    <p style="color: red;">Este parágrafo está em vermelho.</p>
  1. Interno: O CSS é definido dentro de uma tag <style> no cabeçalho do documento HTML.
    <html>
    <head>
        <style>
            p {
                color: green;
            }
        </style>
    </head>
    <body>
        <p>Este parágrafo está em verde.</p>
    </body>
    </html>

  1. Externo: O CSS é escrito em um arquivo separado com extensão .css, e o HTML o referencia.

    3.1. Arquivo CSS (estilos.css):

        body {
        background-color: lightgray;
    }
    h1 {
        color: navy;
    }

    3.2. Arquivo HTML:

        <html>
    <head>
        <link rel="stylesheet" href="estilos.css">
    </head>
    <body>
         <h1>Minha Página</h1>
    </body>
    </html>

Principais funcionalidades do CSS:

  • Cores e fundos: Altera cores de texto, planos de fundo e bordas.
    background-color: yellow; color: black;
  • Fontes e texto: Controla estilos de fonte, alinhamento, espaçamento e decoração.
    font-family: Arial, sans-serif; text-align: center;
  • Layout: Gerencia o posicionamento e o espaçamento de elementos.
    margin: 10px; padding: 20px;
  • Responsividade: Torna as páginas adaptáveis a diferentes tamanhos de tela.
    @media (max-width: 600px) {
        body {
            font-size: 14px;
        }
    }

Por que o CSS é importante?

  • Ele separa o design do conteúdo, facilitando a manutenção.
  • Permite criar páginas modernas, responsivas e visuais atraentes.
  • Dá flexibilidade para personalizar a aparência de qualquer elemento da página.

3. Linguagem JavaScript

Definição:

JavaScript é uma linguagem de programação amplamente usada para tornar páginas da web interativas e dinâmicas. Enquanto HTML define a estrutura e CSS define o estilo, o JavaScript adiciona funcionalidade e comportamento, permitindo que os sites façam coisas como responder a cliques de botões, atualizar conteúdo sem recarregar a página ou até criar animações.

“Hello World” na linguagem de programação JavaScript 

    console.log("Hello, World!");

Principais características do JavaScript:

  1. Dinâmico: Pode alterar elementos de uma página HTML em tempo real, como modificar texto, estilos ou adicionar novos elementos.
  2. Baseado em eventos: Reage a ações do usuário, como cliques, digitação, rolagem ou envio de formulários.
  3. Amplo uso: Embora seja mais famoso no desenvolvimento web, também pode ser usado em servidores (Node.js), aplicativos móveis e até automação.

Como funciona o JavaScript?

O JavaScript é integrado ao HTML e geralmente executado diretamente no navegador do usuário. Ele pode ser adicionado de três formas:

  1. Inline: Escrito diretamente no elemento HTML usando o atributo onclick ou similar:
    <button onclick="alert('Olá!')">Clique aqui</button>
  1. Interno: Inserido em uma tag<script> dentro do HTML:
    <html>
    <head>
        <script>
            function mostrarMensagem() {
                alert("Olá, mundo!");
            }
        </script>
    </head>
    <body>
        <button onclick="mostrarMensagem()">Clique aqui</button>
    </body>
    </html>
  1. Externo: Em um arquivo.js separado, vinculado ao HTML com <script>:

    3.1. Arquivo script.js:

        function mostrarMensagem() {
        alert("Olá, mundo!");
    }

    3.2. Arquivo HTML:

        <html>
    <head>
        <script src="script.js"></script>
    </head>
    <body>
        <button onclick="mostrarMensagem()">Clique aqui</button>
    </body>
    </html>

O que o JavaScript pode fazer?

  • Interatividade:
    • Mostrar ou esconder elementos.
    • Validar formulários (ex.: garantir que o e-mail está no formato correto).

  • Atualização dinâmica:
    • Atualizar partes de uma página sem recarregá-la (ex.: comentários em tempo real).
    • Trabalhar com APIs para buscar dados (ex.: previsão do tempo, cotações de moedas).

  • Efeitos visuais:
    • Animações e transições.
    • Controle de sliders e galerias de imagens.

  • Armazenamento local:
    • Salvar informações no navegador do usuário (ex.: preferências de tema ou dados temporários).

Exemplo básico:

Um contador que aumenta ao clicar em um botão:

    <!DOCTYPE html>
    <html>
    <head>
        <title>Exemplo de JavaScript</title>
        <script>
            let contador = 0;
            function incrementar() {
                contador++;
                document.getElementById("contador").innerText = contador;
            }
        </script>
    </head>
    <body>
        <h1>Contador: <span id="contador">0</span></h1>
        <button onclick="incrementar()">Incrementar</button>
    </body>
    </html>

Por que o JavaScript é importante?

  • Ele transforma páginas estáticas em experiências interativas e envolventes.
  • É amplamente suportado por navegadores e possui uma grande comunidade de desenvolvedores.
  • É a base de várias bibliotecas e frameworks populares, como React, Angular e Vue.js.

4. Linguagem Python

Definição:

Python é uma linguagem de programação de alto nível, versátil e fácil de aprender, usada para uma ampla variedade de aplicações, como desenvolvimento web, automação, análise de dados, inteligência artificial, e muito mais. Criada por Guido van Rossum em 1991, Python é conhecido por sua sintaxe simples e clara, o que a torna uma das linguagens preferidas tanto para iniciantes quanto para programadores experientes.

Principais características do Python:

  1. Simplicidade: Sua sintaxe é projetada para ser legível e intuitiva.
  2. Versatilidade: Pode ser usada para desenvolver aplicativos web, scripts de automação, análise de dados, aprendizado de máquina, entre outros.
  3. Bibliotecas poderosas: Possui milhares de bibliotecas e frameworks para praticamente qualquer tarefa.
    • Exemplo: NumPy para matemática, Pandas para dados, Django para web, TensorFlow para IA.
  4. Interpretada: O código é executado linha por linha, sem necessidade de compilação, tornando mais fácil testar e corrigir erros.
  5. Comunidade ativa: Python tem uma comunidade global que contribui com ferramentas, bibliotecas e suporte.

Por que Python é tão popular?

  • Fácil para iniciantes: Ótimo para aprender lógica de programação sem se preocupar com sintaxe complicada.
  • Multiplataforma: Funciona em Windows, Mac e Linux.
  • Ampla aplicação: Pode ser usado em diferentes áreas, como desenvolvimento web, automação, ciência de dados, robótica, e até jogos.

Exemplo de código em Python:

  1. “Hello World” na linguagem de programação Pyhton
    print("Hello, World!")
  1. Outro exemplo: cálculo de soma
    # Solicita dois números do usuário
    num1 = int(input("Digite o primeiro número: "))
    num2 = int(input("Digite o segundo número: "))
    # Calcula a soma e exibe o resultado
    soma = num1 + num2
    print("A soma é:", soma)

Aplicações do Python:

  1. Automação de Tarefas (Scripts):
    • Exemplo: Renomear arquivos em massa, enviar e-mails automáticos.
  2. Desenvolvimento Web:
    • Usando frameworks como Django e Flask.
  3. Ciência de Dados e IA:
    • Processamento de grandes volumes de dados, criação de modelos de aprendizado de máquina com bibliotecas como TensorFlow e Scikit-learn.
  4. Desenvolvimento de Jogos:
    • Usando bibliotecas como Pygame.
  5. Administração de Sistemas:
    • Ferramentas para monitorar servidores, gerenciar arquivos, entre outros.

Como começar com Python?

  1. Instale o Python:
    • Já mencionou que tem Python instalado. Ótimo! 😊
  2. Escolha uma IDE ou Editor de Código:
    • Pode usar o IDLE (vem com Python), VS Code, PyCharm ou qualquer editor de texto.
  3. Execute um código simples:
    • Crie um arquivo com extensão .py e execute-o no terminal.
       print("Bem-vindo ao Python!")

5. Linguagem PHP

Definição:

O PHP (Hypertext Preprocessor) é uma linguagem de programação interpretada e de código aberto, especialmente projetada para o desenvolvimento de aplicações web. Ele é executado no servidor e permite criar páginas dinâmicas que interagem com bancos de dados, processam formulários e gerenciam sessões de usuários.

“Hello World” na linguagem de programação PHP

    <?php
    echo "Hello, World!";
    ?>

Características principais do PHP:

  1. Linguagem Server-Side: O código é processado no servidor e o resultado é enviado ao navegador como HTML.
  2. Fácil de aprender: A sintaxe é simples e amigável para iniciantes.
  3. Compatível com vários sistemas: Funciona em servidores Linux, Windows, macOS e outros.
  4. Alta integração: Funciona bem com HTML, CSS, JavaScript e a maioria dos bancos de dados, como MySQL, PostgreSQL e SQLite.
  5. Ampla utilização: Muitas plataformas populares, como WordPress, Joomla e Drupal, são construídas em PHP.

Por que usar PHP?

  • Ele é ideal para criar sites dinâmicos e interativos.
  • Suporta diversas funções nativas para acesso a bancos de dados, manipulação de arquivos, autenticação de usuários, entre outros.
  • Tem uma enorme comunidade, com muitos tutoriais, extensões e suporte.

Como funciona o PHP?

  1. O código PHP é escrito em um arquivo com extensão .php.
  2. Quando um cliente (navegador) solicita a página, o servidor executa o PHP.
  3. O servidor gera o HTML resultante e o envia ao navegador.

Exemplo de código PHP:

  • Exemplo 1: "Olá, mundo!" 
        <?php
    echo "Olá, mundo!";
    ?>

    Quando você salva esse código em um arquivo chamado index.php e o executa em um servidor com suporte a PHP, ele exibirá "Olá, mundo!" no navegador.

  • Exemplo 2: Processando um formulário 
        <form method="post" action="processar.php">
        Nome: <input type="text" name="nome">
        <input type="submit" value="Enviar">
    </form>

    Arquivo processar.php:

        <?php
    $nome = $_POST['nome'];
    echo "Olá, " . htmlspecialchars($nome) . "!";
    ?>

Aplicações do PHP:

  1. Desenvolvimento de Sites Dinâmicos:
    • Exibir conteúdos que mudam com base no usuário ou banco de dados.
  2. Sistemas de Gestão de Conteúdo (CMS):
    • Plataformas como WordPress são construídas em PHP.
  3. Interação com Bancos de Dados:
    • Consultas e manipulação de dados com SQL.
  4. Gerenciamento de Sessões e Usuários:
    • Login, autenticação e gerenciamento de sessões.
  5. APIs:
    • Criar e consumir APIs para integração entre sistemas.

Vantagens do PHP:

  • Popularidade: Muitos sites e sistemas web usam PHP, tornando fácil encontrar recursos e suporte.
  • Flexibilidade: Pode ser usado para projetos simples e complexos.
  • Bibliotecas e Frameworks: Ferramentas como Laravel, Symfony e CodeIgniter aceleram o desenvolvimento.

Considerações:

Embora o PHP tenha sido criticado por problemas de segurança e por ser considerado ultrapassado por alguns, ele continua evoluindo e sendo usado em projetos modernos. A versão mais recente do PHP traz melhorias significativas em desempenho e segurança.
